## sample1

```html
<table class="fixed_header">    <!-- display: table;(by default, no need to write) table-layout: fixed;  -->
  <thead>                       <!--  -->
    <tr>                        <!-- display: block;  -->
      <th>Col 1</th>            <!-- width: 200px; -->    
    </tr>                     
  </thead>                      <!--  -->
  <tbody>                       <!--  display:block;overflow: auto;max-height: 150px; -->
    <tr>                        <!--  -->
      <td>row 1-0</td>          <!-- width: 200px; -->
    </tr>
  </tbody>
</table>
```
