## Magicui- Globe+Orbiting Circles+Marquee+Particles

### javascript version

1. `npm i cobe`
2. create 'Globe.jsx' and 'OrbitingCircles.jsx'

### typescript version

- Globe(Magi UI)
  - `npx shadcn@latest add @magicui/globe`
  - `import { Globe } from "@/components/ui/globe"`, `<Globe />`
  - https://magicui.design/docs/components/globe
- OrbitingCircles(Magi UI)
  - `npx shadcn@latest add @magicui/orbiting-circles`
  - `import { File, Search, Settings } from "lucide-react"`
  - `import { OrbitingCircles } from "@/components/ui/orbiting-circles"`
  - https://magicui.design/docs/components/orbiting-circles
- Marquee(Magi UI)
  - `npx shadcn@latest add @magicui/marquee`
  - https://magicui.design/docs/components/marquee
