### 'npm : 无法加载文件 E:\NodeJS\npm.ps1，因为在此系统上禁止运行脚本 【解决办法】'

命令行运行：

- `get-ExecutionPolicy`
- `set-executionpolicy remotesigned`

### Set node path

- `SET PATH=C:\Program Files\Nodejs;%PATH%`
