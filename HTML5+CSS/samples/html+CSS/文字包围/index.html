很多年了，习惯了抬头看天空。当春天的风筝布满天空的时候，笑容里是甜蜜的一汪清泉。当大雁飞过天空的时候，秋天已经到了尽头，笑容里迷离着一股离愁。喜欢抬头的那种姿态，很多关于梦的想法，都会觉得不再遥远。抬起头看天空的那一刻，放飞的心绪里满载着无数的渴望。无论是想放下悲伤，还是想充满希望，就在那遥远的天际，如丝如缕。 很早都学会了不惧怕的活着，因为每天的天空上都会出现太阳，于是不惧怕遇见的困难，是因为明天的太阳不会因为某一种因素而不在天上。仰望天空，收获着力量。 风筝飞起了，我想起过。大雁南飞时，我依然会想起.可能是曾经心动的人，可能是曾经帮助过自己的人，也可能只是一个模糊的情愫，更可能是一个很早的梦想。 深居在秋色里，望着天上的大雁飞过，小小的大雁能越过千山万水的从北方迁移到南方，需要多大的力量来支撑着它们的一路艰辛。不惧怕的精神，让大雁成群结队的，彼此鼓励彼此支持着，一路向南。 我们也像一群大雁，飞翔在人生的天空中。 前面的路程是自己不可以预知的未来，可能或有险滩，可能会有丛林，可能会遇上豺狼虎豹，可能会有生命的危险。这所有的困难，我们都要不惧怕它的存在，而勇往直前。 当你披荆斩棘，当你委屈流泪，当你绝望彷徨，当你得意尽欢，这一切的遭遇都是我们应该遇见的，所以，不要途中放弃，不要自暴自弃。 再次凝望着远去的大雁，秋天随着时间的流逝，渐渐远去。 我们都会老去，可是，在你飞翔的时候，你是否竭尽全力? 当秋天不再，大雁飞过千山，南方的冬天还会冷吗? 秋色连绵处，苍凉有感。却又有无数景色值得欣赏，有菊花之绚烂，有红叶之亮眼，有大雁飞过时留下的声音和美丽的倩影。谁说这不是一个很美很美的秋天呢。雁自南飞，秋在尽头，在秋中，我们走过无数的路。 走长城 一 读小学时，历史课本图片中逶迤于崇山峻岭的长城便给我以雄关漫道的形象。时日迁延，对长城了解愈深，愈是强化了我登临长城的念头，可是一宜没有机会。 今年三月，在同学的介绍下，做了回导游志愿者，带领来自澳大利亚青年发展使团的十二个成员参观长城。我做梦也没想到，第一次走近长城，竟然是和澳大利亚青年男女为伴! 那天起得很早，因为我得从对外经济贸易大学赶往北京语言大学，带上那些澳大利亚青年。大巴行了两个多小时，终于到达金山岭。我们的金山岭——司马台长城之旅历时一天。带着兴奋、激动和些许凝重回到学校，已经夜凉如水了。 二 和那些外宾同游，他们让我讲讲长城的来历。我告诉他们，几千年前，中国出现了多国并存并相互混战的局面，为了防止敌人入侵，燕、赵、秦等国用巨石、砖块泥土在山顶上磊起了长城，并驻以军队加强防御。他们听了，都为中国人的聪明而发出赞叹。 可他们哪里知道，这些筑了长城的国家，一个个都灭了. 大修长城，消耗了大量的人力物力，而在以农业为基础的古代中国，势必造成生产力的降低。国不富，民不强，几块石头围起的城又怎能关得住涣散的民心?古人常云改朝换代乃是天意，实是天下万民之意!历史上那些大兴土木、鱼肉百姓的朝代，哪个逃脱了这道魔咒?!屹立于崇山峻岭之上的长城，沉重地压断了撑起国家的百姓的脊梁。梁既断，大厦又怎能不倾? 长城啊长城，你确实体现了中国建筑艺术的辉煌成就，却也记载了历史当权者的无知、贪婪和民众近似麻木的忍耐! 三 游玩那天，一进门，一大群人背着包跟在了我带的使团后面，有白发的老人，还有中年的妇女。看着那朴素的装束和诚实的面孔，我就断定出他们是农民。起初我想，现在的农民日子好了，还能有时间来逛长城了，我为此很是欣慰。 谁知上了长城之后，这些人快速超过我们，在前面打开包裹，在长城上铺上了地摊，有卖可乐、短袖，也有卖珠串、卡片什么的。见到我带的外国人，他们就开始用单个的英语单词吆喝开了，可这些外国人很少买的。一旦见老外上前了，他们便收了东西，迅速走到前面，再次铺开摊子。在老外面前移动的地摊，让我这个中国导游的脸不觉微微发烫。 和其中的一个中年妇女交谈后，我才知道他们是当地的农民，很多就靠着移动的地摊维持家计。还有一个妇女，因老外讲好了价，却少付了五元钱而求助于我。我问她是不是亏本了，她说没亏，也没赚多少钱。我没有帮她，觉得这样只会让我的脸更红更烫，便劝她别计较了，她听了我的。可是后来我很自责，一直问自己，我为什么那么狠心不去帮她?与她生活的艰难相比，我的面子值几个钱? 四 队伍走的很慢。下午两点左右，大约走了一半。使团里一个胖女孩走不动了，我们坐在石阶上休息时，三个妇女就凑过来了。其中一个对我说她早上没有吃饭，并问我有没有多余吃的，她出钱买。我把仅存的一袋饼干送给了她，还给她一瓶水，她连连道谢。一边走，一边望着眼前身后千峰聚会的山野，我的脑海中一次次浮现出元代才子张养浩的 山坡羊.潼关怀古 ：“峰峦如聚，波涛如怒…兴，百姓苦; 亡，百姓苦。”看看眼前，的确峰峦如聚啊，可惜这里没有黄河如怒的波涛，现在也不是元朝! 这就是长城上特殊的游客 他们不是为兴致而游，而是为了生计，日复一日年复一年，在这冰冷生硬的石头上艰难而又坚强地游走!他们沧桑的面孔、沉鱼的背包、匆忙的脚步，比我国的国内生产总值、外汇储备、报纸报道的人均收入更加鲜活丰满，更为生动形象!我暗问自己：看不见的长城之外，景色会是什么样子? 我越走越饿，直到晚上八点多，才回到学校，在校外的小饭馆内点了一个鸡腿，一份炒饼，一瓶啤酒，尽管很饿，但却是没有吃出滋味，那瓶啤酒倒是让我醉了。我就如此不胜酒力?我想可能是已先醉千长城上那白雪点缀的山峦和频频移动的摊点拼凑起来的山野人文景观吧。
